Setting Up

Prerequisites

Must have custom shortcuts enabled in Settings.app.

Select Your Device

Run the shortcut and follow the initial steps to set the shortcut for your specific device.

Downloading Content.zip

After setting the shortcut to your device, the shortcut will now download a .zip file needed to run the shortcut. The shortcut will prompt you to manually unzip the package in your Files.app.

Layouts

Layout is the term used for an image layer consisting of elements from the Content.zip. A copy of your created layouts is saved and can be used again in the future. Read ahead to understand building a layout more.

Wallpaper

Selecting a wallpaper photo

After you're done with building or choosing a layout, you're now selecting a wallpaper image to use from your Photos.app.

Masking Options

Once a wallpaper is selected, the shortcut will ask you if you'd like to mask your wallpaper. This crops out certain parts of the wallpaper to achieve different effects.
